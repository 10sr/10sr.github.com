git-svn

Subversionリポジトリと連携できるgit-svn | 2ページ | SourceForge.JP Magazine - SourceForge.JP

  1. clone

    $ git svn clone svn+ssh://user@example.com/path/to/rep

  2. edit contents and commit changes

  3. pull changes

    $ git svn rebase

  4. push changes

    $ git svn dcommit

Git で集中リポジトリーに push したら、自動でワーク・ディレクトリーにも反映させる

(cd /path/to/non_bare/rep && git --git-dir=.git pull)
