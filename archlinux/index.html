swap file Swap - ArchWiki

  • # fallocate -l 2G /2g.swap

  • # mkswap /2g.swap
  • # swapon /2g.swap で終了するまでスワップが有効になる
  • /etc/fstab に /2g.swap none swap defaults 0 0 を追記。
