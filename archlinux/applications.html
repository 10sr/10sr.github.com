Xdg-open - ArchWiki

多分普通はpcmanfmで開いたほうが良い

まず、 xdg-mime query filetype a.txt と xdg-mime query default text/plain を調べる。 それがないと、mimeopenを使う。設定はxdg-mimeよりmimeopenのほうがずっと楽。

  1. perl-file-mimeinfoをインストール
  2. パスが通ってないので、ログアウトするかmimeopenを探す。
  3. $ mimeopen -d file.ext で開くプログラムが設定できる。
  4. これ以降は、xdg-openでも上で設定したプログラムで開かれるようになる。
